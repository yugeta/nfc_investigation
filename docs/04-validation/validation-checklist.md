# Android NFCシステム検証チェックリスト

## 機能
- 目標遅延内でタグを検出できる。
- 有効なペイロードで正しい業務アクションが起動する。
- 期限切れまたは不正ペイロードが拒否される。
- 短時間の重複タップが正しく処理される。

## 信頼性
- サポート対象端末間で読み取り成功率が安定している。
- 低品質ネットワーク条件でも挙動が許容範囲内である。
- リトライキューと復旧フローが正しく動作する。

## セキュリティ
- 改ざんペイロードが常に拒否される。
- nonce/TTL方針によりリプレイ試行が遮断される。
- APIが未署名または不正形式リクエストを拒否する。
- 監査ログが完全かつ改ざん不能である。

## 運用
- オペレーターが開発者の支援なしでエラーを理解・復旧できる。
- 監視により異常な失敗増加を検知できる。
- 鍵ローテーションとインシデント対応のランブックがある。

## 受け入れエビデンス
- タイムスタンプと端末モデル付き試験ログ
- KPIサマリーレポート
- 要件とテストケースのトレーサビリティ
